ChordHelp

ChordHelp is a React web app that I wrote for myself. I am learning to play the Guitar, Ukulele, and Banjo and wanted a centralized source for chord information. The web app dynamically displays all of the chords for each instrument, and I can even add alternate versions for the chords if I come across them. It's a practical tool that I've already begun to use when practicing.

Sacramento 311 Mobile App

For my Senior Project at Sac State, I worked with a team to redevelop the Sacramento City 311 Mobile App from the ground up. We developed it for both major mobile platforms with just one codebase, using Expo Router for React Native in order to accomplish this. The project is written in typescript and we used GitHub to collaborate. The app is currently in the hands of the city's IT team, and they intend to use some of the ideas or code from it in the future.
