Sacramento 311 Mobile App

For my Senior Project at Sac State I have been working with a team to redevelop the Sacramento City 311 Mobile App from the ground up. We are developing it for both major mobile platforms, while only using one codebase. We are using Expo Router for React Native in order to accomplish this. We are writing in typescript and using GitHub to collaborate. We do have a functioning prototype, however I have only shown our mockups here as it is still in progress.
